# 포트폴리오 웹사이트

Next.js와 Tailwind CSS를 활용하여 제작한 개발자 포트폴리오 웹사이트입니다.

## 🚀 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Font**: Geist (Google Fonts)
- **Deployment**: Vercel (권장)

## ✨ 주요 기능

- 📱 반응형 디자인
- 🌙 다크 모드 지원
- ⚡ 빠른 로딩 속도
- 🎨 현대적인 UI/UX
- 📄 SEO 최적화

## 🛠️ 설치 및 실행

### 1. 프로젝트 클론

```bash
git clone [your-repository-url]
cd elfinlas
```

### 2. 의존성 설치

```bash
npm install
# 또는
yarn install
# 또는
pnpm install
```

### 3. 개발 서버 실행

```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

## 📁 프로젝트 구조

```
src/
├── app/
│   ├── globals.css          # 전역 스타일
│   ├── layout.tsx           # 루트 레이아웃
│   └── page.tsx             # 홈페이지
├── components/              # 재사용 가능한 컴포넌트 (추후 추가)
└── lib/                     # 유틸리티 함수 (추후 추가)
```
